# Data Analysis With Anomaly Focus Examples

## Example 1

User ask:

`What looks unusual in this spreadsheet?`

Example output:

- Observed anomaly: Returns spike sharply in one region during the last period.
- Evidence: The latest value is far above the recent baseline and is not mirrored in other regions.
- Scope or affected segment: The anomaly is concentrated in the South region only.
- Why it matters: This one spike is large enough to distort overall margin performance.
- Confidence or uncertainty: Medium confidence because the recent history is short.

## Example 2

User ask:

`Find the weird parts of this table for me.`

Example output:

- Observed anomaly: One product line shows a sudden drop in completion rate while others remain stable.
- Evidence: The decline is abrupt and appears only in the last two periods.
- Scope or affected segment: The issue is isolated to a single line rather than the full dataset.
- Why it matters: It may indicate a process break or data quality issue.
- Confidence or uncertainty: Medium to high confidence because the deviation is large and localized.
