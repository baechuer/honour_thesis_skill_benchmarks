# Data Analysis Overview Examples

## Example 1

User ask:

`Give me a quick read on this spreadsheet so I can see what matters most.`

Example output:

- Overall picture: Revenue is growing steadily, but margin is softening in the most recent period.
- Notable patterns: Two product lines drive most of the growth, while returns are concentrated in one region.
- Main concern or opportunity: The current growth pattern looks healthy, but the return spike may erode gains if it continues.
- Key takeaway: The business appears to be growing, but quality and profitability signals are mixed.

## Example 2

User ask:

`Summarise the main patterns in this table for me.`

Example output:

- Overall picture: Performance is uneven across categories rather than uniformly improving.
- Notable patterns: The top segment is stable, the middle segment is volatile, and the bottom segment is declining.
- Main concern or opportunity: The middle segment may be the best place to investigate because it shows the largest swings.
- Key takeaway: The dataset suggests a concentrated performance problem rather than a universal one.
