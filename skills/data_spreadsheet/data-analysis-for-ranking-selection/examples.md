# Data Analysis For Ranking Selection Examples

## Example 1

User ask:

`Rank these vendors for me based on cost, delivery reliability, and defect rate.`

Example output:

- Ranking or recommended option: Vendor B looks strongest overall, followed by Vendor A, then Vendor C.
- Main reasons: Vendor B has the best reliability profile while staying competitive on cost.
- Tradeoffs or caveats: Vendor A is cheaper, but its defect rate is less stable.
- What would change the decision: If cost is weighted much more heavily, Vendor A could become preferable.

## Example 2

User ask:

`Which applicants should I shortlist from this sheet?`

Example output:

- Ranking or recommended option: The strongest shortlist is Priya Nair, Lucas Chen, and Hannah Lee.
- Main reasons: They combine the highest scores on relevant experience, availability, and interview quality.
- Tradeoffs or caveats: Lucas has slightly weaker availability than the others.
- What would change the decision: If immediate start date becomes the dominant criterion, Hannah would move ahead.
