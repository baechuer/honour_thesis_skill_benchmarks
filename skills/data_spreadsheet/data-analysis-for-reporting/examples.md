# Data Analysis For Reporting Examples

## Example 1

User ask:

`Turn this spreadsheet analysis into something I could send to a manager.`

Example output:

- Topic: Weekly sales performance
- Main findings: Sales increased overall, but growth was concentrated in two categories and offset by weaker margin in one region.
- Supporting observations: Category A and B account for most of the uplift, while returns increased noticeably in the South region.
- Practical implication or recommendation: Growth looks real, but the return pattern should be reviewed before scaling the current approach.

## Example 2

User ask:

`Write a short report-style summary from this dataset.`

Example output:

- Topic: Applicant screening outcomes
- Main findings: Most shortlisted candidates cluster in the mid-to-high score range, with one criterion driving most exclusions.
- Supporting observations: Availability and prior experience explain more of the ranking spread than certification alone.
- Practical implication or recommendation: The current filter may be overly strict on one requirement and should be checked before final selection.
