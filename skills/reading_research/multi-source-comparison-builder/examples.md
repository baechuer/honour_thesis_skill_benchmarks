# Multi-Source Comparison Builder Examples

## Example 1

User ask:

`Compare these two papers on agent planning so I can see how their methods and assumptions differ.`

Example output:

- Comparison dimensions: planning strategy, tool use, evaluation setting, limits
- Source A: Explicit hierarchical planning with staged execution
- Source B: Direct iterative planning with lightweight revision
- Similarities: Both evaluate multi-step agent behavior and report gains over direct baselines.
- Differences: Source A uses stronger decomposition and more structured control; Source B is lighter-weight and more flexible.
- Key takeaway: The papers address similar problems but make different tradeoffs between structure and adaptability.

## Example 2

User ask:

`Show me how these two sources differ on evidence and scope.`

Example output:

- Comparison dimensions: evidence type, scope, claims, limitations
- Source A: Empirical benchmark study with quantitative results
- Source B: Conceptual discussion of retrieval ambiguity
- Similarities: Both focus on agent performance under complex task settings.
- Differences: Source A provides measured outcomes; Source B offers a framing argument and design implications.
- Key takeaway: The sources complement each other, but they contribute different kinds of evidence.
