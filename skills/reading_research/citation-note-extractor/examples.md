# Citation Note Extractor Examples

## Example 1

User ask:

`Turn this paper into citation notes I can use later in my literature review.`

Example output:

- Main point: The paper argues that tool-using agents perform better when task decomposition is explicit.
- Useful supporting detail: The authors evaluate on multi-step benchmark tasks and report higher completion reliability under decomposed workflows.
- Possible use in writing: Can support a claim that explicit task structure improves multi-step agent performance.
- Limitation or caution: Evidence is benchmark-specific and may not generalize to noisier open-ended environments.
