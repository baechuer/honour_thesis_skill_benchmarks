Recent work on tool-using agents splits broadly between approaches that emphasize explicit decomposition and approaches that rely more on iterative refinement during execution. ## Example 2

User ask:

`Write a short related-work synthesis from these sources on skill retrieval and agent planning.`

Example output:

Existing work on skill-based agents and planning-oriented systems points to a shared concern with reliability under complex task demands. Some sources emphasize retrieval and routing quality, while others focus on decomposition and execution control. Together they suggest that strong downstream behavior depends not only on model capability but also on how intermediate structure is represented and selected. A remaining gap is how these mechanisms behave under realistic semantic overlap rather than idealized task separation.
