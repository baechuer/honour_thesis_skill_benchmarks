# Document Rewriter Examples

## Example 1

User ask:

`Rewrite this internal note so it sounds clearer and more professional, but keep the commitments the same.`

Example output:

`We are still on track to send the revised budget on Friday. The current draft needs one final check on vendor totals before it goes out. If any figures change, we will flag them in the cover note rather than sending a separate update.`

## Example 2

User ask:

`Make this policy paragraph easier to read without changing what it requires.`

Example output:

`Requests must be submitted at least five working days in advance. Approval depends on staff availability and budget sign-off. If supporting documents are missing, the request may be delayed until the file is complete.`
