# Document Converter Examples

## Example 1

User ask:

`Convert this meeting note into markdown bullets I can drop into my repo.`

Example output:

`## Meeting notes

- Budget review moved to Friday
- Slides need one final edit
- Open question: whether to include the appendix in the shared version`

## Example 2

User ask:

`Turn this form-like text into a short memo.`

Example output:

`Memo

The request covers laptop replacement for two staff members. Estimated cost is AUD 3,400. Approval is still pending finance review, and purchase should not proceed until sign-off is complete.`
