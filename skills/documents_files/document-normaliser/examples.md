# Document Normaliser Examples

## Example 1

User ask:

`Clean this note dump into something more consistent without changing the actual content.`

Example output:

`Project update

- Draft report is complete
- Finance section still needs one figure checked
- Send revision to the team on Tuesday

Risks

- Vendor confirmation still missing
- Timeline may slip if approval is delayed`

- Submit requests at least five working days in advance
- Include the purpose of travel and estimated cost

Approval

- Manager approval is required first
