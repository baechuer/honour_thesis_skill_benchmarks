# Email Polisher

Improves an existing email draft as a full communication artifact while preserving its core intent.

## Use when

- The user already has a draft email.
- The goal is to improve clarity, professionalism, flow, tone, or persuasive effectiveness.
- The user wants the full email strengthened before sending.
- The draft is a professional, academic, administrative, or workplace email rather than a short direct reply.

## Not for

- Polishing a short direct reply where preserving the original response almost line-for-line is the main concern.
- Writing a brand-new email from notes or thread context only.
- Summarizing an email thread.
- Extracting actions or deadlines without drafting an email.
- Changing the email's core commitments or meaning unless explicitly asked.

## Workflow

1. Read the existing draft first.
2. Identify the intended purpose, commitments, requests, and recipient expectations.
3. Improve wording, structure, readability, and tone at the whole-email level.
4. Reorganize paragraphs if needed to make the email clearer or more effective.
5. Remove awkward phrasing, redundancy, and unnecessary hedging.
6. Keep the final email aligned with the original purpose and recipient.

## Writing rules

- Preserve the core purpose, commitments, dates, and requests unless the user asks to change them.
- It is acceptable to restructure the email more substantially than a light reply polish if that improves clarity or effectiveness.
- Avoid adding new claims not grounded in the existing draft or source context.
- Return only the polished draft unless the user asks for commentary or alternatives.

## Distinguishing emphasis

This skill is for improving a full professional email as a complete document. It may:

- rewrite the opening to set context more clearly
- reorder paragraphs to improve flow
- tighten the request or call to action
- improve subject-line clarity if one exists or is needed

It should not be treated as a light-touch reply editor for short response messages. That narrower case belongs to a reply-polishing skill.
